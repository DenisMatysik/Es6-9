 Objects

 set и get - свойств которые в объекте работают как методы а снаруэи как свойства.

Своййства objects (es7-9)
entries - врзвращает массив из массивов собственных перечесляемых свойст объекта в формте ключ-значение
(если объект наследует свойтсва от прототипов то эти свойтсва отображатсья не будут). Своййство ключом которого
являеться символ(Symbol) игнорируется.
fromEntries - возвращает объект из коллекции пар типа ключ-значение.
values - врзвращает массив из собственных значение, без ключей (так же исключаются значения из цепочки прототипов).

getOwnPropertyDescriptors() - возвращает все сведеия для все свойств заданного объекта, включая данные о гетерах и сетерах.
Данный метод позволяет создавать мелке копии объектов и клонировать объекты создавая новые, при этом копирую гетеры и сетеры.
В основном используеться для копирования объектов:
const person = {
  name: "Max",
  age: 30,
  set personName(name) {
    this.name = name;
  },
  get password() {
    return `${this.name}${this.age}`;
  },
};

const superAdmin = Object.defineProperties(
  {},
  Object.getOwnPropertyDescriptors(person)
);
