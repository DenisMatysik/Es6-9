Set/Map коллекции для хранения данных.
Map - коллекция для хранения данных вида ключ - значение, в отличии от объекта в map ключом могут быть произвольные значения.
Для добавления элемента в коллекцию используется метод set, для получения - get. Чтобы найти длину коолекции есть метод size.
Чтобы проверить еслть ли элемент в коллекции есть метод has. Для удаления записи по ключу используют метод delete, 
для очистки всей коллекции есть метод clear.
Set коллекция для хранения уникальных данных.
Для добавления значений в коллекцию set используют метод add и с помощью чейнинга добавляют все нужные значения.
Перебор коллеции осуществляетсья с помощью метода for или for ... off.
