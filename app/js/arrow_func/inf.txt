Arrow function
() => {}
Стрелочные функции нельзя использовать как конструкторы объектов(нелльзя импользовать с оператором New).
В стрелочных функциях нет псевдомассива arguments.
Со стрелочными функциями нельзя применять bind, aply, call; т.к. в стрелочных функциях нельзя изменять значение this,
т.к. оно автоматически берёться из контекста.
